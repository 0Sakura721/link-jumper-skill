## 核心概念

link-jumper 是一套通用跨平台消费方法论。核心理念：不预设具体平台或设备，AI理解的是方法论。

**三种场景：**
- 打开链接 — AI分析协议→调用系统Intent或命令行
- 搜索内容（无API）— AI通用搜索→解析→跳转
- 播放视频 — AI提取流地址→本地播放
